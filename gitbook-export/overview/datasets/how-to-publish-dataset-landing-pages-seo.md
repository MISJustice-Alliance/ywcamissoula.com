# How to publish dataset landing pages (SEO)

## How to publish dataset landing pages (SEO)

Google ranks pages. It also understands “collections” better when they have a single landing page.

That landing page is your dataset. Treat it like a mini abstract + scope statement.

### What to do

1. Create **one landing page per collection**.
2. Put the **core keywords** in the first 2–3 lines.
3. Link to the underlying index pages and files.
4. Keep the URL stable. Avoid renames.
5. Update the landing page when scope changes.

### Practical SEO notes (non-technical)

* Use consistent naming: “Police reports dataset”, “Court filings dataset”, etc.
* Prefer one collection per page. Avoid mixing unrelated categories.
* Add internal links from analysis pages back to the relevant dataset page.
* Put “Missoula”, “Montana”, “Seattle”, “Edmonds”, and key agencies in scope lines when relevant.
