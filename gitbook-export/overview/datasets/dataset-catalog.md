# Dataset catalog

## Dataset catalog

This section creates **dataset landing pages** for major record collections.

Use these pages when you want one URL that describes a collection, its scope, and how to cite it.

### Datasets

* [Primary records dataset](primary-records-dataset.md) (police reports, court documents, correspondence)
* [Official complaints dataset](official-complaints-dataset.md) (state and federal filings)
* [Timeline & claims map dataset](timeline-and-claims-map-dataset.md) (events, relationships, and claim framing)

### How this helps Google

* These pages give each collection a **single canonical URL**.
* They repeat **scope + coverage** in a consistent way.
* They concentrate internal links to primary sources.

Use [How to publish dataset landing pages (SEO)](how-to-publish-dataset-landing-pages-seo.md) as the template.

Quick export: [SEO meta snapshot](seo-meta-snapshot.md).

{% hint style="info" %}
Google “Dataset” results usually require machine-readable structured data. These landing pages still help regular Search by making the collection explicit and linkable.
{% endhint %}
